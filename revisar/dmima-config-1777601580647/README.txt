==============================================
D'Mima - Archivos de Configuracion
==============================================

Fecha de exportacion: 1/5/2026, 4:13:00

ARCHIVOS INCLUIDOS:
-------------------

1. js/dados.js
   - Contiene todos los productos del menu
   - Reemplace el archivo original en la carpeta /js/

2. js/app-config.js
   - Contiene la configuracion de WhatsApp, direccion, municipios y categorias
   - Copie las variables al inicio de su archivo app.js original

3. index-referencias.html
   - Contiene las referencias actualizadas de contacto
   - Use los valores para actualizar su index.html original

COMO APLICAR LOS CAMBIOS:
-------------------------

1. Haga una copia de seguridad de sus archivos originales
2. Reemplace js/dados.js con el archivo incluido
3. Copie las variables de app-config.js al inicio de su app.js
4. Actualice los enlaces de contacto en index.html segun las referencias
5. Suba los archivos modificados a su servidor
6. Recargue la pagina para ver los cambios
